## Share a conversation

Open a session → click the **Share** button → choose a mode.

> [!NOTE]
> Screenshot: share dialog with link and password options.

### Access

| Option | Behavior |
|---|---|
| Public link | Anyone with the URL can view |
| Password-protected | Viewer must enter the password |

### Update mode

| Mode | Behavior |
|---|---|
| Live | Recipients see new messages as they arrive |
| Frozen | Snapshot at the moment of sharing — later messages are not visible |

## Organize the sidebar

- **Pin / Unpin** — keep important sessions at the top of the sidebar
- **Star** — bookmark a session for later

## Incognito mode

Start a session in incognito when you don't want it kept. Incognito sessions are not saved to history and are automatically deleted after 24 hours.
