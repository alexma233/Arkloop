Appearance is purely visual. Nothing here affects agent behavior.

> [!NOTE]
> Screenshot placeholder: Appearance settings with theme preview.

## Theme mode

- System: follow the OS preference
- Light: force light
- Dark: force dark

Path: `Settings -> Appearance -> Theme`

## Font

- Body font: applied to chat, settings, and most UI text
- Code font: applied to code blocks, tool output, and terminals

## Font size

Compact / Normal / Relaxed. Affects line height and padding density across the app.

## Color schemes

Pick a built-in palette or define your own:

- Default
- Terra
- GitHub
- Nord
- Catppuccin
- Tokyo Night
- Retina Burn
- Custom

## Custom color editor

Edit accent colors used for buttons, links, and highlights. Changes preview live before saving.

## Background image

Upload an image and adjust opacity. Used as the chat backdrop.

## Theme import / export

Export the current theme as a portable file, or import one shared by someone else.
