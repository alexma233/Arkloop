title: Settings Overview
description: A map of every settings category.
---

Settings are grouped into focused categories. Open them from the sidebar at `Settings`.

> [!NOTE]
> Screenshot placeholder: Settings sidebar with all categories.

| Category | What it does |
|---|---|
| General | Language, timezone, startup behavior, notifications |
| Appearance | Theme, fonts, color schemes, background |
| Providers | LLM provider connections and model management |
| Channels | Telegram, Discord, Feishu, QQ, WeChat integration |
| Plugins | Installed plugins and marketplace |
| Skills | Skill packages from ClawHub or local sources |
| MCP | Model Context Protocol server configuration |
| Notebook | Manual notes injected into every conversation |
| Memory | Auto-organized semantic memory |
| Chat | Auto-compaction and execution mode |
| Tools | Web search and fetch provider configuration |
| Advanced | Data import/export, network, usage stats, logs, modules |

Most categories are independent. Memory and Notebook are two separate subsystems that can be enabled together or alone. Modules under Advanced power optional capabilities like sandboxed code execution and semantic memory.
