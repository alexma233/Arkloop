Arkloop 桌面端由三个主要区域组成：侧边栏、对话区和右侧面板。

## 侧边栏

左侧区域，管理所有会话。

- **会话列表**：两种视图模式
  - 时间线：按时间排列全部会话
  - GTD：按状态分类 — Backlog / Next / Waiting / Someday / Archived
- **置顶区域**：常驻显示重要会话
- **项目分组**：按项目归集会话
- **搜索**：快速查找历史会话
- **新建会话按钮**：创建新对话

## 对话区

中间主区域，包含输入栏和消息流。

**输入栏**

- Persona 选择器：切换 Agent 角色
- Model 选择器：切换使用的模型
- 文本输入框
- Slash 命令：`/plan` 生成计划、`/setup` 调整配置
- 文件附件
- 语音输入
- Reasoning Effort：调节推理深度

**消息气泡**

- 对话轮次展示
- Cop Timeline：思考块、工具调用、来源引用
- 特殊卡片：TodoList、CodeExecution、SubAgent、Artifact、MemoryAction

## 右侧面板

可切换的辅助面板，三种模式：

| 面板 | 功能 |
|------|------|
| Browser | 网页预览、前进后退、书签 |
| Files | 本地文件浏览 |
| Resource | Markdown / 代码 / 文档预览 |

## 设置入口

侧边栏或顶栏的齿轮图标，点击进入设置面板。所有配置项均可在此修改：Provider、主题、语言、记忆、Agent 等。
