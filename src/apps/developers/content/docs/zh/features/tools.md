Arkloop 的 Agent 通过工具完成任务——可调用的能力模块，涵盖文件读写、网页搜索、代码执行等。工具来自三个来源：无需配置即可使用的内置工具、通过 MCP 服务器连接的外部工具、以及扩展 Agent 专业能力的技能包。

## 内置工具

内置工具对所有 Agent 默认可用，按用途分组。

### 文件操作

| 工具 | 用途 |
|------|------|
| `read` | 读取文件内容 |
| `write_file` | 创建或覆盖文件 |
| `edit` | 对已有文件进行精确编辑 |
| `glob` | 按名称模式查找文件 |
| `grep` | 按内容模式搜索文件 |

### 网页

| 工具 | 可用提供商 |
|------|-----------|
| `web_search` | Basic / Tavily / Exa / SearXNG |
| `web_fetch` | Basic / Jina / Firecrawl |

搜索提供商在 设置 → Providers 中配置。抓取提供商决定网页内容的获取和解析方式。

### 代码执行

| 工具 | 用途 |
|------|------|
| `sandbox` | 在隔离环境中执行代码 |

沙箱提供安全的一次性容器，用于运行用户或 Agent 生成的代码，输出和错误信息会返回到对话中。

### 任务管理

| 工具 | 用途 |
|------|------|
| `todo_write` | 创建或更新任务清单 |
| `enter_plan_mode` | 切换到计划模式——先列出步骤再执行 |
| `exit_plan_mode` | 返回执行模式 |

### 子 Agent

| 工具 | 用途 |
|------|------|
| `spawn_agent` | 将子任务委派给并行运行的子 Agent |

子 Agent 在对话中以 SubAgentBlock 形式展示，从派生到完成全程可见。

### 用户交互

| 工具 | 用途 |
|------|------|
| `askuser` | Agent 在任务中途向用户请求输入 |
| `show_widget` | 在对话中渲染交互组件 |

### 其他

| 工具 | 用途 |
|------|------|
| `summarize_thread` | 生成当前对话的摘要 |
| `arkloop_help` | 查询 Arkloop 产品文档和帮助信息 |

## MCP 工具

MCP（Model Context Protocol）工具是通过 MCP 服务器连接的第三方能力。默认不可用——需要通过 设置 → MCP 添加。

MCP 服务器配置完成后，其暴露的所有工具自动对 Agent 可用。Agent 根据对话上下文自主决定何时调用。

配置方法参见 [MCP 设置](/docs/settings/mcp)。

## 技能包

技能是可安装的工具包，为 Agent 增加新能力。它们将提示词、工具、有时还有数据打包为一个整体。

### ClawHub 市场

浏览并安装社区发布的技能。涵盖从专用工具到完整工作流集成的各类技能。

### 本地技能

从 GitHub 仓库导入技能，或直接从文件系统上传。

### 内置技能

Arkloop 自带的技能：

| 技能 | 能力 |
|------|------|
| `opencli` | OpenCLI 命令执行 |
| `geogebra-drawing` | GeoGebra 交互数学图形 |

技能安装后，Agent 自动获得对应的工具能力，无需手动配置。

## 平台管理

`platform_manage` 工具仅对平台 Agent 可用，提供 40+ 管理操作，覆盖项目、Persona、线程、计费和管理功能。普通 Agent 无法使用此工具。
