对话界面是 Arkloop 的核心交互区域，包含消息流、输入栏和辅助面板。本文覆盖全部功能点，从基础对话到高级操作。

## Chat / Work 双模式

Arkloop 提供两种对话模式，适应不同的交互强度。

| | Chat | Work |
|---|---|---|
| 交互方式 | 一问一答 | Agent 自主多步执行 |
| 适用场景 | 日常对话、问答 | 编程、调研、长流程任务 |
| 进度展示 | 无 | 进度面板和任务追踪 |

Chat 模式适合快速问答和日常对话，每轮交互由用户驱动。Work 模式适合需要多步规划与执行的任务，Agent 会自主拆解目标、逐步推进，右侧展示进度面板和任务追踪。

切换方式：点击输入栏上方的模式按钮 → 选择 Chat 或 Work。

## 输入栏

输入栏位于对话区底部，是发送消息和配置运行参数的主入口。

### Persona 与 Model 选择

点击输入栏上方的 PersonaModelBar → 切换 Agent 人格或模型。Persona 决定 Agent 的行为风格和工具集，Model 决定底层推理引擎。

### Slash 命令

在输入框中输入 `/` 触发命令菜单：

- `/plan` → 进入计划模式，Agent 先生成执行计划再动手
- `/setup` → 打开项目设置

### 文件附件

两种方式向 Agent 传递文件内容：拖拽文件到输入栏，或点击附件按钮选择文件。附件内容会作为上下文注入当前对话。

### 语音输入

点击麦克风按钮进行语音输入。该功能需在设置中配置 ASR 凭证后方可使用。

### Reasoning Effort

控制 AI 的思考深度，六档可选：Off / Minimal / Low / Medium / High / Max。档位越高，AI 花费越多 token 进行内部推理，回答更周全但耗时更长。日常对话选 Low 或 Medium 即可，复杂推理任务可拉到 High 或 Max。

### 自动调整高度

文本输入框会随内容自动扩展高度，无需手动拖拽。

## 消息操作

每条消息支持一系列快捷操作，悬停或右键即可触发。

### 复制与编辑

- **复制消息** → 将消息内容复制到剪贴板
- **编辑消息** → 修改已发送的消息内容并重新发送，后续对话基于新内容重新生成

### 反馈与收藏

- **报告消息** → 标记问题类型：不准确 / 过时 / 太短 / 太长 / 有害 / 来源错误
- **Star / Unstar** → 收藏消息，方便后续查找
- **Pin / Unpin** → 置顶消息，在长对话中快速定位关键内容

### 会话管理

- **隐身模式** → 开启后不保存历史记录，24 小时后永久删除
- **分享会话** → 生成分享链接，他人可查看对话内容
- **重命名会话** → 给会话设置有意义的名称
- **删除会话** → 彻底移除会话及其消息

### 中断与重试

- **中断回复** → Agent 正在生成时点击停止，截断当前输出
- **重试回复** → 对已有回复重新生成，获取不同结果

## Cop Timeline

AI 回复中嵌入的执行过程展示，让 Agent 的每一步操作透明可见。

### Thinking 块

显示 Agent 的思考过程。包含思考时长和动画提示语，帮助理解 Agent 正在分析什么。点击可展开查看完整思考内容。

### 工具调用展示

Agent 调用外部工具时展示执行状态：

- **Shell 执行** → 显示命令和成功/失败状态
- **Web Fetch** → 显示抓取的网页内容
- **代码执行** → 显示沙箱中的代码运行结果

### Source 列表

回复引用的外部来源，点击可查看原始内容。帮助验证回答的可靠性。

### 展开 / 收起

Timeline 各区块默认折叠，点击展开查看细节。长回复中可有效控制信息密度。

## 特殊卡片

对话中出现的富内容卡片，承载结构化信息展示。

| 卡片 | 用途 |
|------|------|
| TodoListCard | 任务进度追踪，显示各步骤的完成状态 |
| CodeExecutionCard | 沙箱代码执行结果，含 stdout 和 stderr |
| SubAgentBlock | 子 Agent 状态：派生中 / 运行中 / 完成 / 失败 / 关闭 |
| UserInputCard | Agent 向用户请求输入，等待确认后继续 |
| MemoryActionBlock | 记忆操作展示：写入 / 搜索 / 读取 / 删除 |
| ContextCompactBar | 上下文压缩进度，显示 token 使用情况 |

### Artifact 预览

Agent 生成的富内容可实时预览：

- **HTML** → 浏览器级渲染
- **SVG** → 矢量图形渲染
- **图片** → 直接展示
- **Stream** → 流式内容渲染

### 图表类卡片

- **MermaidBlock** → Mermaid 语法生成的流程图、序列图等
- **MindmapBlock** → 思维导图，展开层级结构
- **GeoGebraBlock** → GeoGebra 数学图形，支持交互操作

## 右侧面板

对话区右侧的辅助面板，三种模式按需切换。

| 面板 | 功能 |
|------|------|
| Browser | 网页预览，含前进后退和书签功能 |
| Files | 本地文件浏览，快速查看项目文件 |
| Resource | 资源预览，支持 Markdown / 代码 / 文档渲染 |

Browser 面板在 Agent 执行 Web Fetch 时自动切换显示。Files 面板可浏览 Agent 工作区中的文件。Resource 面板用于预览消息中引用的各类资源。

## 侧边栏

左侧会话管理区域，提供多种组织和检索方式。

### 视图模式

- **时间线模式** → 按时间倒序排列全部会话
- **GTD 模式** → 按状态分类：Backlog / Next / Waiting / Someday / Archived

### 组织方式

- **置顶区域** → 常驻显示重要会话，不受视图模式影响
- **项目分组** → 按项目归集会话，快速定位项目相关对话

### 快捷操作

- **搜索** → 在历史会话中快速查找
- **新建会话** → 创建空白对话开始新任务
