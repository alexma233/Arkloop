---
title: 设置总览
description: 所有设置分类的导航地图。
---

设置按职责分组，从侧边栏 `Settings` 进入。

> [!NOTE]
> 截图占位：设置侧边栏全貌。

| 分类 | 用途 |
|---|---|
| General | 语言、时区、启动行为、通知 |
| Appearance | 主题、字体、配色、背景 |
| Providers | LLM 服务商连接和模型管理 |
| Channels | Telegram、Discord、飞书、QQ、微信接入 |
| Plugins | 已装插件和插件市场 |
| Skills | 来自 ClawHub 或本地的 Skill 包 |
| MCP | Model Context Protocol 服务器配置 |
| Notebook | 注入到每次对话的手写笔记 |
| Memory | 自动整理的语义记忆 |
| Chat | 自动压缩与执行模式 |
| Tools | 网络搜索和抓取的服务商配置 |
| Advanced | 数据导入导出、网络、用量统计、日志、模块 |

各分类基本独立。Memory 和 Notebook 是两个独立子系统，可单独使用也可叠加。Advanced 中的模块为可选能力，例如沙箱代码执行和语义记忆。
