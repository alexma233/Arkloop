General 控制 Arkloop 如何启动、如何与系统协作、何时打扰你。

> [!NOTE]
> 截图占位：通用设置面板。

## 语言和时区

界面语言独立于 agent 使用的语言。时区用于聊天记录、定时触发器和用量图表的时间戳。

路径：`Settings -> General -> Language` / `Timezone`

## 启动行为

- 开机自启：系统登录后自动启动 Arkloop
- 启动时打开窗口：是否显示主窗口，否则常驻托盘

## 关闭行为

- 后台保持运行：关窗后运行时不退出，心跳和频道 bot 保持在线
- 关闭即退出：关窗时关闭所有 sidecar

## 桌面通知

长任务结束时弹出。在 agent 调用工具反复迭代时切走标签页很有用。

## 更新提醒

有新版 Desktop 时提示，不自动安装。

## 活跃期间保持屏幕常亮

run 进行中阻止系统休眠，默认关闭。
