外观仅影响视觉，不会改变 agent 行为。

> [!NOTE]
> 截图占位：外观设置与主题预览。

## 主题模式

- System：跟随系统
- Light：强制浅色
- Dark：强制深色

路径：`Settings -> Appearance -> Theme`

## 字体

- 正文字体：用于聊天、设置和大多数 UI 文本
- 代码字体：用于代码块、工具输出和终端

## 字号

Compact / Normal / Relaxed，影响全局行高和留白密度。

## 配色方案

选内置预设或自定义：

- Default
- Terra
- GitHub
- Nord
- Catppuccin
- Tokyo Night
- Retina Burn
- 自定义

## 自定义颜色编辑器

调整按钮、链接、高亮所用的强调色，保存前实时预览。

## 背景图

上传图片并调整不透明度，作为聊天背景。

## 主题导入 / 导出

把当前主题导出为可移植文件，或导入他人分享的主题。
